# NI ARC Agricultural Autonomous Robot (LabVIEW-Based Control System)

## Overview
This project was developed for the National Instruments Autonomous Robot Competition (NI ARC). The system focuses on designing and implementing an agricultural-based autonomous robot using a LabVIEW software control approach.

## Objective
To develop a robot capable of performing agricultural-related tasks using sensor integration, real-time control, and graphical programming in LabVIEW.

## System Architecture
- Control System: NI LabVIEW
- Sensors: IR
- Actuators: Planetary Geared Motors,Servo Motors
- Microcontroller / NI Hardware: MyRIO-1900

## Key Features
- Autonomous navigation
- Sensor-based obstacle detection
- Real-time monitoring using LabVIEW interface
- Modular system design for scalability
